This file maps projects to the hackathon tracks, sponsor awards, and merit
badges we are targeting.

## Merit Badges

| Badge | Practical Target |
| --- | --- |
| Off the Grid | Local runtime or GGUF apps without third-party inference APIs |
| Well-Tuned | Fine-tuned adapter published on Hugging Face |
| Off-Brand | Custom UI beyond default Gradio styling |
| Llama Champion | GGUF model running through llama.cpp |
| Sharing is Caring | Public dataset, traces, or training outputs on Hugging Face |
| Field Notes | Build report with lessons, architecture, and results |
